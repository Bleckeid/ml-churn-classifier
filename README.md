# ☕ Прогнозирование оттока клиентов сети кофеен

## Описание проекта
Разработка модели машинного обучения для предсказания оттока клиентов (churn). На основе анализа поведения пользователей и истории их заказов модель выявляет тех, кто с большой вероятностью перестанет пользоваться услугами.

## Основные этапы работы
1. **Предобработка данных:** Работа с пропусками, фильтрация аномалий в частоте заказов.
2. **Исследовательский анализ (EDA):** Анализ корреляций (Spearman, Phik) и распределений признаков.
3. **Обработка признаков:** Кодирование категориальных данных (TargetEncoder, OneHotEncoder) и масштабирование (StandardScaler).
4. **Обучение модели:** Построение пайплайна (Pipeline) с использованием логистической регрессии.
5. **Валидация:** Кросс-валидация со стратификацией для борьбы с дисбалансом классов.

## Результаты
Модель показала высокую стабильность и качество предсказаний:
* **PR-AUC на тестовой выборке:** `0.746`
* **F1-score:** `0.621`

**Ключевые инсайты:**
* Наибольшее влияние на отток оказывают технические сбои в приложении (количество крашей).
* Клиенты с PRO-подпиской требуют отдельного внимания, так как их поведение значительно отличается от обычных пользователей.
